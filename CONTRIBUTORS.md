## How to Be Added

To be added to this list:

1. **Code Contributions**
   - Submit a pull request that gets merged
   - You'll be automatically added after first merged PR

2. **Documentation**
   - Improve or add documentation
   - Fix typos or clarify instructions

3. **Bug Reports**
   - Report valid bugs with clear reproduction steps
   - Significant bugs will be credited

4. **Testing**
   - Participate in beta testing
   - Provide valuable feedback

## Become a Contributor

We're always looking for contributors! Here's how you can help:

1. **Check open issues**: Look for issues labeled `good first issue` or `help wanted`
2. **Read CONTRIBUTING.md**: Understand our contribution guidelines
3. **Fork and code**: Make your changes and submit a PR
4. **Join discussions**: Participate in GitHub Discussions

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines.
